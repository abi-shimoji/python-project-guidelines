# Python 開発ガイドライン

このディレクトリは、Python プロジェクトの開発運用を揃えるためのガイドラインを管理するためのものです。

- `PYTHON_GUIDELINES.md`: Python 開発時の標準ツールと運用方針
- `RUFF_LINT_GUIDE.md`: `ruff` の lint 運用ガイド
- `COMMENT_GUIDE.md`: コメントと docstring の運用ガイド

## Python プロジェクトテンプレート

`uv`、`ruff`、`ty`、`pytest` を前提にした最小構成の Python プロジェクトテンプレートは、以下のリポジトリにあります。

https://github.com/abi-techinnovation/python-project-template

新規プロジェクトを始めるときはテンプレートリポジトリを利用し、運用ルールはこのリポジトリの各ガイドを参照します。
