# python-project-guidelines

このリポジトリは、Python プロジェクトを立ち上げるためのテンプレートと、開発運用を揃えるためのガイドラインを管理するためのものです。

- `template/`: `uv`、`ruff`、`ty`、`pytest` を前提にした最小構成のプロジェクトテンプレート
- `PYTHON_GUIDELINES.md`: Python 開発時の標準ツールと運用方針
- `RUFF_LINT_GUIDE.md`: `ruff` の lint 運用ガイド

新規プロジェクトを始めるときは `template/` をベースにし、運用ルールは各ガイドを参照します。
